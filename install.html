Radiant – Business analytics using R and Shiny

How to install Radiant

Radiant is a platform-independent browser-based interface for business analytics in R, based on the Shiny package. Developed by Vincent Nijs. Please use the issue tracker on GitHub to suggest enhancements or report problems: https://github.com/vnijs/radiant/issues. For other questions and comments please use radiant@rady.ucsd.edu.

Note: Profiles (or usernames or logins) containing non-ASCII characters will cause problems during the install. To address this issue please use a profile name (login) which contains ASCII (A-E, a-e) Letters only. For example, c:\users\myprofilename

1. Install R for your system

For mac go to https://cran.rstudio.com/bin/macosx/

For windows you can get the latest release here: https://cran.rstudio.com/bin/windows/base/release.htm

Important: Windows users should install R to directory C:\R\R-3.2.2

On Mac you can accept all the defaults. Installing R should create 2 shortcuts on your Desktop. If there are indeed 2 shortcuts you can remove the icon named (“R i386 3.2.2”).

2. Install Rstudio (OPTIONAL)

Get the latest release of Rstudio for your system from:

https://www.rstudio.com/products/rstudio/download/

3. Use a modern browser

For example, Chrome or Safari. Internet Explorer (version 11 or higher) should work as well. Chrome, however, is the recommended browser.

4. Install Radiant

To install Radiant open R (or Rstudio) and copy-and-paste the command below. Then press return. R (Rstudio) will now install Radiant and several dependencies. This may take a few minutes depending on the speed of your internet connection.

Note: On Windows you may need to start R (or Rstudio) by right-clicking on the desktop shortcut and choosing run as administrator.

install.packages("radiant", repos = "http://vnijs.github.io/radiant_miniCRAN/", type = "binary")

If you are asked where to install Radiant and its dependencies (i.e., “Would you like to use a personal library instead?”) answer “yes” or “y” and then press return.

5. Create a desktop launcher

To create icons on your Desktop to start and update Radiant, copy-and-paste (or type) ONE of the commands below into R (or Rstudio). For the Quantitative Analysis app use the quant command, for the Marketing Research app use the marketing command, and for the Customer Analytics app use the analytics command:

## Choose one of the following three commands depending on the class you are in
radiant::launcher("quant")

radiant::launcher("marketing")

radiant::launcher("analytics")

After pasting the command press return. When asked if you want to create a short-cut type y and press return. A file called radiant.bat (windows) or radiant.command (mac) will be created that you can double-click to start Radiant in your default browser. The launcher command will also create a file called update_radiant.bat (windows) or update_radiant.command (mac) that you can double-click to update Radiant to the latest release.

Alternatively, you can start Radiant from R (or Rstudio) using one of the commands below. For the Quantitative Analysis app use quant, for the Marketing Research app use marketing, and for the Customer Analytics app use analytics:

library(radiant)

## Choose one of the following three commands depending on the class you are in
radiant("quant")

radiant("marketing")

radiant("analytics")

When Radiant starts you will see data on diamond prices. To close the application click the icon in the navigation bar and then click Stop. The Radiant process will stop and the browser window will close (Chrome) or gray-out.

If you get stuck, take a look at the Installing Radiant video:

Documentation

Documentation and tutorials for each of the apps are available at http://vnijs.github.io/radiant/ and in the Radiant web interface (the icons on each page and the icon in the navigation bar).

Want more help getting started? Watch the tutorials on the documentation site.

Trouble shooting

As mentioned above, Windows users may need to start R (or Rstudio) by right-clicking on the R (or Rstudio) desktop shortcut and choosing run as administrator.

On Windows, if you see a message such as “Warning: unable to move temporary installation ‘C:-library.254ad976f6’ to ‘C:-library.2’” you may need to reinstall R into another directory (e.g., C:\R\R-3.2.2). You should then first remove the directory Documents\R from your computer

On rare occasions R(studio) may complain that a package is not yet installed. If so, install the package mentioned using the command below but replace ‘package-that-is-not-yet-installed’ with the package name mentioned in the error message

install.packages("package-that-is-not-yet-installed", repos = "http://vnijs.github.io/radiant_miniCRAN/")

If R(studio) complains about not being able to delete a package or directory please copy-and-paste the commands below into R (or Rstudio) and press return. Then double-click the update_radiant icon on your desktop again. If the update is successful (i.e., no error messages) try double clicking the radiant icon on your desktop again.

remove.packages("Rcpp", lib = .libPaths())
remove.packages("dplyr", lib = .libPaths())

Reporting issues

Rady students can report any issues they might have with Radiant on the class Piazza site.

Otherwise, please use the GitHub issue tracker at github.com/vnijs/radiant/issues if you have any problems with Radiant.
