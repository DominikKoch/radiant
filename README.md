## Radiant: Business analytics using R and Shiny

Interactive business analytics using [R](http://www.r-project.org/) and [Shiny](http://www.rstudio.com/shiny/). Developed by <a href="http://rady.ucsd.edu/faculty/directory/nijs/" target="_blank">Vincent Nijs</a>. You can reach me at radiant@rady.ucsd.edu. Full source code for Radiant is available on <a href="https://github.com/radiant-rstats" target="_blank">GitHub</a>.
